# Checklist of the Butterflies of Alaska


This repository is for maintaining a checklist of the butterflies of
Alaska in Darwin Core format. These data are intended to provide a
taxonomic framework for a potential update to the *Butterflies of
Alaska* ([Philip and Ferris 2015](#ref-PhilipFerris2015),
[2016](#ref-PhilipFerris2016)).

The structure of the checklist follows guidance of GBIF
([2017](#ref-GBIF2017)). The `archive/taxon.csv` table follows the
format of the [Darwin Core
Taxon](https://rs.gbif.org/core/dwc_taxon_2025-07-10.xml) extension.
